# 图片颜色聚类可视化

基于自实现 K-Means++ 算法的像素色彩分析与 ECharts 可视化展示工具。纯前端单文件，CDN 引入 ECharts 后即可在浏览器中直接运行。

## 功能概览

### 基本功能

- **K-Means 聚类**：完全自主实现，采用 K-Means++ 初始化 + 多轮 SSE 选优策略，结果稳定可复现
- **像素颜色均值可视化**：以真实色块展示每个类簇的聚类中心颜色
- **像素数量可视化**：饼图/柱状图展示各类簇像素占比与数量
- **ECharts 图表**：环形饼图与柱状图，支持一键切换

### 进阶功能

| 功能 | 说明 |
|------|------|
| 交互式 K 值 | 滑块控件，范围 2～15 |
| 图片选择 | 点击上传或拖拽，支持 JPG / PNG / WebP |
| 可视化样式切换 | 饼图 ↔ 柱状图 |
| 颜色空间切换 | RGB / LAB（D65 标准光源） |
| 颜色名称映射 | 内置 170+ 中文颜色名称（含中国传统色、莫兰迪色系），通过加权最近邻匹配 |
| AI 和谐度判断 | 集成 DeepSeek API，由大模型评估聚类颜色的搭配和谐度并打分 |
| 莫兰迪 UI | 暖米色低饱和配色，响应式布局 |

## 快速开始

1. 下载 `index.html`
2. 用浏览器直接打开
3. 点击上传区选择一张图片
4. 调整 K 值滑块，点击「运行聚类分析」
5. （可选）点击「判断颜色和谐度」调用 AI 评估

> 无需安装任何依赖，无需启动服务器。

## 技术实现

### K-Means 聚类

```
1. K-Means++ 初始化中心点
2. 将每个像素归为距离最近的类簇
3. 根据新划分重新计算中心位置
4. 迭代至收敛（中心位移 < 0.1 或达到最大迭代次数）
5. 重复 1-4 共 8 轮，选取 SSE 最低的结果
```

### 颜色空间转换（RGB ↔ LAB）

- sRGB 线性化 → CIE XYZ（D65）→ CIE LAB
- 支持在 LAB 空间进行聚类以获得更符合人眼感知的结果

### 颜色名称映射

内置 **170+ 中文颜色名称数据库**，覆盖红/橙/黄/绿/青/蓝/紫/棕/灰/莫兰迪/中国传统色等色系。使用加权欧氏距离（人眼对绿色最敏感 ×4，蓝色最弱 ×3）进行最近邻匹配。

### AI 评测

调用 `deepseek-chat`（DeepSeek API）模型，以专业色彩搭配设计师视角评估聚类所得颜色组合的和谐度，输出评分与分析。
